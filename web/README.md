Preparing node
--------------------------------------------------------------------------------

  * `npm install --global express --save`
  * `npm install --global angular`
  * `npm install --global react`
  * `npm install -g create-react-app`
    (Vermutlich nicht erforderlich!)
  * `npm install --global pug --save` (Template engine ehemals Jade)
